# 🌍 AI Translator App - Pinnacle Internship (Task2)

## 📌 Objective

This project translates text from one language to another using AI-based Natural Language Processing techniques.

## 📊 Input

User-provided text for translation into selected target language.

## ⚙️ Technologies Used

* Python
* Streamlit
* Deep Translator

## 🤖 Functionality

* Real-time text translation
* Multiple language support
* Automatic language detection

## 🌐 Web App

A Streamlit web application is created for instant translation of user input text.
